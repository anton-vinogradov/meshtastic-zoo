# meshtastic-zoo 📡

Карта зоопарка Meshtastic-нод: две площадки (`10.77.77.0/24` и `10.88.88.0/24`),
внешний мир с резервными нодами и качество радиолиний (SNR) между ними.

Статичный сайт без сборки и зависимостей: HTML + CSS + vanilla JS, рендер в SVG.

## Посмотреть

Открыть `index.html` в браузере, либо:

```sh
python3 -m http.server 8814
# → http://localhost:8814
```

## Обновить данные

Вся топология — в одном файле [`data/topology.js`](data/topology.js):

- `nodes` — ноды: `zone` (подсеть или `world`), `x`/`y` — позиция в долях зоны;
- `links` — линки: `lan` (провод внутри площадки), `bridge` (межплощадочный
  радиомост), `rf` (радиоплечо; `snr` в dB, `null` + `note` = не измерено,
  `labelT` — положение подписи вдоль линии 0..1);
- `meta.snrScale` — шкала «% от идеала»: `floor` дБ → 0% (красный),
  `ideal` дБ → 100% (зелёный); цвет плеча — непрерывный градиент по этому
  проценту, подпись — SNR, точный % — в тултипе плеча.

Наведение на ноду подсвечивает её линки и соседей.
